# Smart Parking System using Arduino

An embedded system that automates parking slot detection and gate control using ultrasonic sensors.

## Features
- Detects occupancy of 3 parking slots
- Displays available slots on LCD
- Controls entry gate using servo motor
- LED indicators for slot availability

## Hardware Used
- Arduino Uno
- Ultrasonic Sensors (HC-SR04)
- Servo Motor
- 16x2 LCD (I2C)
- LEDs

## System Logic
Car arrives → Entry sensor checks distance → If slot available → Gate opens → LCD updates.
